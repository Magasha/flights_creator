# Скрипт создания рейсов — yamagistrali.ru

Скрипт читает Excel файл с заказами и автоматически создаёт рейсы через API Яндекс Магистралей с назначением внешнего исполнителя ("Повезу не я").

---

## Требования

- Python 3.9+
- Токен Яндекс OAuth

---

## Установка

```bash
pip install -r requirements.txt
```

---

## Настройка (config.py)

Откройте `config.py` и заполните:

```python
YANDEX_OAUTH_TOKEN = "y0_AgAAAA..."   # ваш OAuth токен
EXECUTOR_ID        = "12345"           # ID вашей организации-исполнителя
EXCEL_PATH         = "orders_template.xlsx"  # путь к вашему Excel файлу
```

### Как получить OAuth токен

1. Откройте в браузере:
   ```
   https://oauth.yandex.ru/authorize?response_type=token&client_id=<APP_ID>
   ```
   (APP_ID уточните у администратора платформы или в личном кабинете yamagistrali.ru)
2. Войдите в Яндекс аккаунт, связанный с платформой
3. Скопируйте токен из URL после редиректа (`access_token=...`)

### Как узнать EXECUTOR_ID

Запустите диагностику:
```bash
python debug_api.py --cargo-id <любой_номер_заказа>
```
В блоке `[2]` будет выведен список доступных ID для вашего токена.

| Стоимость с НДС | ✅ | Число (например `12000`) |

## Запуск

```bash
python main.py
```

Скрипт:
1. Прочитает Excel файл
2. Найдёт заказы в системе через API
3. Покажет превью — сколько заказов найдено, что будет создано
4. Дождётся нажатия **Enter** для подтверждения
5. Создаст рейсы

## Диагностика

Если заказы не находятся — запустите:

```bash
python debug_api.py --cargo-id 0-MPPEE-2606-11-15-001
```

Скрипт покажет:
- Сколько задач вообще доступно вашему токену
- Найден ли конкретный заказ по номеру
- Список доступных EXECUTOR_ID для вашего токена

### Типичные причины проблем

| Симптом | Причина | Решение |
|---|---|---|
| Задач найдено: 0 | Неверный `EXECUTOR_ID` | Проверьте блок `[2]` в debug_api.py |
| Заказ не найден среди N задач | Неверный номер заказа | Проверьте написание в Excel |
| HTTP 403 | Токен не имеет прав | Проверьте токен и права доступа |
| HTTP 401 | Токен истёк | Получите новый токен |

## Архитектура

```
config.py        — настройки (токен, EXECUTOR_ID, путь к Excel)
api_client.py    — HTTP-клиент для API yamagistrali.ru
excel_reader.py  — чтение и валидация Excel файла
main.py          — основной скрипт
debug_api.py     — диагностика
requirements.txt — зависимости Python
orders_template.xlsx — шаблон Excel
```

### Как работает поиск заказов

1. `getForFlightAddForExecutor(executorId)` — получает все задачи, доступные исполнителю для добавления в рейс
2. Сопоставляем `taskName` из API с номером заказа из Excel
3. Используем `taskId` для добавления в рейс через `addBulk`

Это официальный API-flow для роли **исполнитель**.

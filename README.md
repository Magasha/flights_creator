# Скрипт создания рейсов — Яндекс Магистрали

Автоматически создаёт рейсы в [yamagistrali.ru](https://yamagistrali.ru) по данным из Excel файла.

**Что делает скрипт:**
1. Читает список заказов и данных по внешним исполнителям из Excel
2. Проверяет, что заказ находится в статусе «В работе» (`onExecute`)
3. Проверяет, что у заказа ещё нет рейса
4. Создаёт рейс
5. Добавляет заказ в рейс
6. Включает «Повезу не я» — назначает внешнего исполнителя с указанием: наименования компании, ИНН, типа компании, страны, валюты, налоговой ставки, стоимости без и с НДС

## Установка

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Получите OAuth2 токен Яндекса

Токен нужен для авторизации всех запросов к API Яндекс Магистралей.

**Способ 1 — через Яндекс OAuth (рекомендуется):**
1. Откройте в браузере:
   ```
   https://oauth.yandex.ru/authorize?response_type=token&client_id=<APP_CLIENT_ID>
   ```
   > `client_id` — это ID приложения, зарегистрированного в Яндекс OAuth для Магистралей.
   > Уточните его у технической поддержки yamagistrali.ru или в разделе «Доступ к API» в вашем аккаунте.

2. Войдите под своим Яндекс аккаунтом, дайте разрешение.
3. Скопируйте токен из URL-адреса после `access_token=`.

**Способ 2 — через Яндекс ID (если доступно):**
- Войдите в [oauth.yandex.ru](https://oauth.yandex.ru/), найдите выданные токены своему приложению.

Токен выглядит как: `y0_AgAAAA...` (длинная строка).

---

### 3. Узнайте ваш Forwarder ID

Это идентификатор вашей организации-экспедитора в системе Магистралей.

```bash
# Временно вставьте токен в config.py, затем выполните:
python3 -c "
from config import YANDEX_OAUTH_TOKEN, BASE_URL
from api_client import MagistraliClient
client = MagistraliClient(YANDEX_OAUTH_TOKEN, BASE_URL)
print(client.get_allowed_forwarders())
"
```

Скрипт выведет список доступных организаций с их ID.

---

### 4. Заполните config.py

Откройте файл `config.py` и заполните:

```python
YANDEX_OAUTH_TOKEN = "y0_AgAAAA..."   # ваш токен
FORWARDER_ID = "12345"                # ваш Forwarder ID
```

## Подготовка Excel файла

Используйте шаблон `orders_template.xlsx` — скопируйте его и заполните своими данными.

### Обязательные колонки

| Колонка | Допустимые названия | Описание |
|---|---|---|
| ID заказа | `id заказа`, `номер заказа`, `order_id`, `id` | Номер заказа в Яндекс Магистралях |
| Название рейса | `название рейса`, `рейс`, `flight_name` | Произвольное название создаваемого рейса |
| Наименование компании | `наименование компании`, `компания`, `company_name` | Название внешнего исполнителя |
| ИНН | `инн`, `inn`, `тин`, `tin` | ИНН компании-исполнителя |
| Тип компании | `тип компании`, `org_type`, `правовая форма` | ООО, ИП, АО и т.д. |
| Страна | `страна`, `country`, `код страны` | Код страны: RU, KZ, BY |
| Валюта | `валюта`, `currency` | Код валюты: RUB, USD, EUR |
| Налоговая ставка | `налоговая ставка`, `ндс %`, `ндс`, `vat` | Ставка НДС в %: 20, 10, 0 |
| Стоимость без НДС | `стоимость без ндс`, `цена без ндс`, `price` | Число (например 83333.33) |
| Стоимость с НДС | `стоимость с ндс`, `цена с ндс`, `price_with_vat` | Число (например 100000) |

> **Регистр не важен.** Колонка может называться `ИНН` или `инн` — скрипт распознает оба варианта.

### Необязательные колонки

| Колонка | Допустимые названия | Описание |
|---|---|---|
| Плательщик НДС | `плательщик ндс`, `is_vat_payer` | Да/Нет (по умолчанию: Да) |
| Контактное лицо | `контактное лицо`, `фио`, `contact_name` | ФИО |
| Телефон | `телефон`, `phone`, `contact_phone` | Телефон |
| Email | `email`, `почта`, `contact_email`, `e-mail` | Email |
| Комментарий | `комментарий`, `comment`, `примечание` | Доп. комментарий |

## Запуск

### Основной запуск

```bash
python main.py --file orders.xlsx
```

### Указать другой лист Excel

```bash
python main.py --file orders.xlsx --sheet "Мой лист"
```

По умолчанию скрипт ищет лист **«Рейсы»**.

### Тестовый запуск (без реального создания рейсов)

Режим `--dry-run` проверяет данные, авторизацию и статусы заказов, но **не создаёт рейсы**:

```bash
python main.py --file orders.xlsx --dry-run
```

Рекомендуется всегда запускать сначала с `--dry-run`.

### Указать Forwarder ID через аргумент (без изменения config.py)

```bash
python main.py --file orders.xlsx --forwarder-id 12345
```

## Типичные ошибки и их решение

| Ошибка | Причина | Решение |
|---|---|---|
| `Токен не настроен` | Не заполнен `YANDEX_OAUTH_TOKEN` в config.py | Вставьте OAuth токен |
| `FORWARDER_ID не указан` | Не заполнен `FORWARDER_ID` в config.py | Узнайте ID (см. шаг 3) и заполните |
| `Лист 'Рейсы' не найден` | Другое название листа в Excel | Передайте `--sheet "Название листа"` |
| `Не найдены обязательные колонки` | Опечатка в заголовке | Проверьте названия колонок по таблице выше |
| `[HTTP 403]` | Нет прав доступа | Проверьте токен и права аккаунта |
| `[HTTP 400]` | Неверные данные запроса | Проверьте значения в Excel: тип компании, код страны, валюту |
| `Заказ имеет статус 'В черновиках'` | Заказ ещё не в работе | Убедитесь, что заказ принят и находится в статусе «В работе» |
| `У заказа 1001234 уже есть рейс` | Рейс уже создавался ранее | Это не ошибка — скрипт пропускает такие заказы |

## FAQ

**Q: Что делать, если заказ есть в Excel, но скрипт его не находит?**  
A: Убедитесь, что ID заказа указан точно так, как он выглядит в интерфейсе Магистралей (без пробелов и лишних символов).

**Q: Можно ли обрабатывать несколько заказов в одном рейсе?**  
A: В текущей версии скрипт создаёт **один рейс на один заказ**. Если вам нужно несколько заказов в одном рейсе — дополните Excel колонкой группировки и обратитесь к разработчику.

**Q: Скрипт создал рейс, но внешний исполнитель не назначился. Что делать?**  
A: Посмотрите в лог — там будет точная ошибка от API. Чаще всего причина: неверный код страны или тип компании. Проверьте значения по справочнику выше.

**Q: Как узнать актуальный список допустимых значений (страны, валюты, типы)?**  
A: Выполните GET запрос к соответствующему справочнику (ссылки в разделе «Справочник значений»). Можно это сделать прямо из браузера, указав в заголовке `Authorization: OAuth ВАШ_ТОКЕН`.
